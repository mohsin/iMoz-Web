Developed a comprehensive agriculture platform supporting smallholder farmers in Colombia with expert knowledge and real-time meteorological data. The platform (primarily in Spanish) provides personalized technical advice on Good Agricultural Practices.

## Backend & Plugins

Built an OctoberCMS/Laravel backend with multiple specialized plugins including weather integration via the aWhere API, farming manuals, a Q&A system, favorites management, user management, and language support. Integrated the aWhere weather API for real-time meteorological data including precipitation, temperature, and crop-specific recommendations.

## Mobile App

Engineered a mobile app (Rowan) using Ionic and Angular for iOS and Android with offline capabilities and Firebase Cloud Messaging for push notifications. Implemented user authentication, location-based services, and OAuth2 integration.

## Features

- Personalized agricultural advice based on soil analysis and weather forecasts
- Farming manuals for different crops and crop planning tools
- Offline capability for areas with limited connectivity
- WhatsApp integration for day-to-day communication
- Push notifications for time-sensitive agricultural updates
