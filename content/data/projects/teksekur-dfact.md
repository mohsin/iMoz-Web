Coded web app components for TekSecur's DFact platform, a DevSecOps solution that helps development teams accelerate deployments and reduce costs.

## Platform Capabilities

The product enables users to do one-click deployment of containerized apps, improve CI/CD pipeline performance, and cost-optimize infrastructure usage. Built using Vue with PrimeVue components and Pinia for state management.
