title: MediScan
---

A highly secure online repository for medical records — X-rays, MRI scans, CT scans, prescription history, patient details, and even surgical instruments.

## How It Works

Each item (called a "prop") is associated with a unique prop ID. A simple QR code sticker identifies the prop. Scanning it fetches the appropriate data and displays it correctly on the app — regardless of whether it's a scan, a document, or a physical instrument.

Since the database is globally accessible, doctors can retrieve patient data from anywhere in the world using any camera-equipped mobile device.

## Tech Stack

Built with an Android app and OctoberCMS/Laravel web backend exposing a RESTful API for the mobile client.
