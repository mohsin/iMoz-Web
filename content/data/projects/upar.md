Built a fun app that gamifies golf into a contest between players.

## Play Modes

The app has 3 modes:

- **Band mode** — players choose a target band to aim for
- **Call the Shot** — players set their own target for a more personal challenge
- **Virtuoso mode** — the app suggests a target based on player skill, designed for professional golfers

Built for Android using Kotlin and RxAndroid.
