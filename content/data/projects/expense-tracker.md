---
title: Expense Tracker for Google Home
---

Track expenses using Google Actions — a voice-driven expense logging skill for Google Home.

## How It Works

Say something like "Track 50 rupees for lunch" and the assistant logs the expense. Built with Dialogflow for natural language understanding and a PHP webhook for data handling.

## Features

- Conversational expense logging via voice commands
- Spending category management
- Expense history queries
