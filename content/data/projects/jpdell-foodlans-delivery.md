Developed a comprehensive food delivery platform using Ionic and Angular for cross-platform iOS and Android compatibility.

## Dual Role Apps

Built two apps in one — one interface for customers to browse restaurants, place orders, and track deliveries; another for delivery personnel to receive orders and navigate using Google Maps.

## Features

- Real-time location tracking via geolocation APIs and Google Maps
- Firebase backend for user management, order processing, and data persistence
- Payment gateway integration for seamless transaction processing
- Push notifications for order status updates and delivery alerts
- In-app rating system and customer feedback for quality assurance
