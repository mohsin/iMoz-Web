Worked on a web app for a scholarship program connected to UpGrad, run by Ronnie Screwvala. The project came through a group called Indiez.

## What Was Built

Built a course catalog, student application management system, scholarship eligibility tracking, and admin dashboard for program coordinators. Used OctoberCMS/Laravel with a Material Design Lite frontend.

## Project History

The project was left due to delayed and reduced payments from Indiez despite completing every milestone on time with full client ratings. This was the same group involved in the Yahavi project, with similar payment issues on that engagement.
