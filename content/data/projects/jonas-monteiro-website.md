Built a modern, production-ready DJ portfolio website showcasing Jonas Monteiro's career with cutting-edge AI capabilities.

## AI-Powered Event Discovery

Engineered a sophisticated LangGraph-powered AI assistant enabling multi-turn conversational interactions for event discovery, venue queries, and music genre filtering. Implemented custom LangGraph workflows for intelligent intent detection, automatic event filtering by date/location/genre, and an integrated FAQ knowledge base with secure prompt injection protection.

Created intelligent natural language search with timezone-aware date parsing (understanding "new years", "this Friday", "weekend"), multi-venue locality filtering, and city-based searches with alternate spelling support. Built a multi-model AI fallback system with quota management and conversation context persistence for seamless user interactions.

## CMS & Content Management

Developed a comprehensive TinaCMS admin interface allowing Jonas to manage all content without technical knowledge — 80+ events with dates, venues, and guestlist settings, photo galleries, video aftermovies, news articles, social media links, FAQ knowledge base, venue information, and site configuration.

Integrated automated recurring event processing with GitHub Actions and a comprehensive test suite for reliable event data.

## Performance & Deployment

Implemented static site generation with Nuxt 4 for blazing-fast performance, optimized image delivery with WebP conversion, and deployed on Netlify for global CDN, serverless functions, and zero-downtime deployments.
