---
title: Mobile Plugin
---

An OctoberCMS plugin for managing mobile application listings across iOS, Android, and other platforms from your CMS backend.

## Features

- Add iOS, Android, and other store app listings to the backend
- Monitor individual app installs by adding a small snippet of client-side code
- Manage multiple apps from a single admin interface

Useful for companies that want to track and promote their mobile apps alongside their website content.
