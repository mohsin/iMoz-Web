Built both the Android and web app from the ground up, including a RESTful API for the two to communicate.

## What It Does

TellUs allows restaurants to collect feedback from customers and provides analytics and insights into the restaurant's performance. The app is localized so that local restaurants can easily use it.

## Web Backend

Built a robust multi-user hierarchy so restaurant owners, sales executives, outlet managers, and others can log in and manage their restaurant feedbacks. They can view statistics and insights on how they're performing and what customers think of them.

## Android App

Wrote a pragmatic RESTful API in the web application so that the Android app (also built by me) integrates seamlessly. Built using reactive programming patterns with RxAndroid and Retrofit for efficient networking.
