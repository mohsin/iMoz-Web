Developed a portfolio website using the shared designs for mobile, tablet, and desktop screen sizes on Figma. Used that to build a pixel-perfect, Nuxt.js based client-side rendered (CSR) web application that deploys to Netlify from a headless git-backed CMS.

## Headless CMS Setup

Content can be added and updated using an administrative section without requiring dedicated hosting for the app, making it very versatile to manage a portfolio without the additional overhead of a recurring annual hosting charge.

## Planner Page

Additionally, added a Planner page with a dynamically generated QR code so people could purchase the planner directly — linking to a UPI payment flow for seamless mobile checkout.
