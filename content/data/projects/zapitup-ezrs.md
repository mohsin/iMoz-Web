Built a comprehensive WooCommerce CRM integration system for EzrsGold's RuneScape gaming platform, handling both OSRS/RS3 virtual currency trading and the EZRSAccounts hand-trained account marketplace.

## WooCommerce Plugins

Developed two custom WordPress plugins enabling real-time synchronization between WooCommerce storefronts and the Bitcat CRM backend. Engineered order synchronization supporting three product types — Sellable (gold/accounts), Buyable (player-to-platform), and Swapable (account trades) — with automatic processing on order creation and updates.

## REST API & Admin Tools

Implemented a REST API endpoint for remote order status control with basic authentication, including status mapping, order note management with author tracking, and CRM-driven inventory updates. Created an admin interface with CRM credentials configuration, order sync toggles, and manual synchronization triggers.

## Multi-Store Architecture

Architected a multi-store management system supporting distinct storefronts for EzrsGold (gold trading) and EZRSAccounts (account sales) with store renaming, creation functionality, and ID-based order linkage preservation across store migrations. Built a complex database schema for multi-store coordination, account tracking, and order references across both platforms.
