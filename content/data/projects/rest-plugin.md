An OctoberCMS plugin that auto-generates RESTful API controllers, eliminating boilerplate when building API endpoints for third-party integrations.

Define your model and the plugin scaffolds the full CRUD controller, authentication handling, and response formatting — enabling rapid API development on top of OctoberCMS.
