title: GenUI - Deepfake Detection Platform
---

Engineered a comprehensive deepfake detection platform with scalable microservices architecture deployed on Kubernetes and GCP.

## ML Model Integration

Implemented advanced ML model integration for video, audio, and image analysis detecting deepfakes with confidence scoring. The system examines visual inconsistencies, structural artifacts, synthetic motion patterns, face inconsistencies, audio signals, metadata, and file structure.

## Backend Services

Built Flask-based microservices for authentication, file upload, report generation, and HubSpot CRM-integrated ticketing system. Architected an asynchronous processing pipeline using RabbitMQ with job queuing and progress estimation. Implemented Google Cloud Storage for file management and PostgreSQL for persistent data storage.

## Frontend & Reporting

Built a Next.js frontend with file dropzone interface for multimedia uploads and report visualization supporting JSON, CSV, XLS, and XLSX export formats.

## Infrastructure

Configured Kubernetes deployments with horizontal and vertical autoscaling for handling variable workloads. Deployed using Terraform Infrastructure as Code with complete CI/CD pipelines.
