---
title: Php Boilerplate
---

A simple-to-use PHP framework inspired by Mathias Bynens' html5-boilerplate. Minimal, with a straightforward configuration file and abstractions over several PHP methods and third-party plugins to make projects server-independent.

## Features

- Quick install similar to WordPress' famous 5-minute setup
- Frontend initialized with HTML5 Boilerplate for a clean starting point
- OOP abstractions for common PHP patterns
- Configuration-driven, server-agnostic design

Successfully used to build several projects before discovering Laravel had already solved this more elegantly — and switched.
