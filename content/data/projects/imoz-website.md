This website — built and continuously maintained as an open-source project to showcase my projects, speaking engagements, and professional history.

## Stack

Built with Nuxt 3 and Vue 3 using Tailwind CSS for styling, `@nuxt/content` for markdown-driven project data, and nuxt-icon for iconography.

## Features

- Masonry project grid with section filtering
- Custom events timeline with photo carousels
- Dark/light mode toggle
- Downloadable resume page
- Static site generation for fast global delivery

## Claude Code Skills

Includes custom Claude Code skills for AI-assisted content workflows:

- **event-import** — automates importing speaking events from local archives: photo curation and WebP conversion, Keynote export, web research, and YAML data entry
- **event-reorganize** — reorganizes event archive folders on SSD to a canonical structure
