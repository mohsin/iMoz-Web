---
title: Barossa Australia Tourism Platform
---

Engineered a comprehensive digital platform for Barossa Valley — one of Australia's oldest and most prestigious wine regions — connecting travelers with accommodations, wine experiences, and regional attractions.

## ATDW Integration

Built a complete ATDW (Australia's Tourism Data Warehouse) API integration with daily synchronization, complex field mappings, and automated change tracking for 1,000+ property listings. Engineered advanced filtering and pagination systems handling date ranges, category sorting, and URL structure normalization across multiple tourism data sources.

## LearnDash LMS Migration

Migrated the existing LearnDash system to consolidate with the main platform, enabling wine certification management and course delivery. Preserved all user accounts and course content throughout the migration.

## AWS Infrastructure

Architected scalable AWS infrastructure using Terraform with Auto Scaling, spot instances, and optimized PHP-FPM configuration. Developed automated backup systems, intelligent cron scheduling, and complete CI/CD deployment pipelines to handle seasonal tourism fluctuations.
