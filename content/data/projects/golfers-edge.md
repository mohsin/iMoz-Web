---
title: Golfer's Edge Booking Website
---

Built a full-stack online booking system for Golfer's Edge indoor golf with end-to-end functionality — from slot selection through payment to calendar sync.

## Booking Engine

Engineered a sophisticated booking wizard supporting 30-minute time slot management, player count optimization, and real-time Google Calendar synchronization to prevent double-bookings. Features pending booking management workflows and a mobile-first responsive design.

## Payments & Invoicing

Implemented Razorpay payment processing with an advanced coupon/discount system, GST-aware invoicing (18% rate handling), and automated confirmation/cancellation emails.

## Admin Dashboard

Developed a comprehensive admin dashboard with multiple reporting widgets including revenue overview with trend analysis, bookings by user sourcing, today's bookings overview, and historical booking trends for business intelligence. Built a club management system with working hours configuration, Google Calendar linking, and role-based access control for club managers.

## AWS Infrastructure

Architected AWS infrastructure using Terraform with EC2, RDS MariaDB, S3 storage, multi-zone VPC setup, CloudWatch monitoring, and automated backup systems. Implemented a CI/CD pipeline with GitHub Actions for continuous deployment and booking source tracking for customer acquisition analytics.
